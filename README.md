# 项目简介

## 核心功能
- 提供基于YOLOv8的数字检测系统完整源码
- 包含4200张数字数据集，涵盖13类（'0'到'9'，'l'，'undefined'，'x'）
- 提供训练好的模型及训练结果
- 支持多种输入方式：单张图片、图片文件夹、视频文件、本地摄像头
- 支持自定义置信度和IOU阈值
- 提供实时显示功能：目标位置、目标总数、置信度
- 支持批量检测并查看所有结果
- 提供检测结果保存功能
- 支持选择不同检测模型
- 提供完整项目运行教程

## 适用场景
- 数字识别与检测应用开发
- 计算机视觉研究与教学
- 自动化检测系统构建
- 视频内容分析处理

## 技术特点
- 基于YOLOv8目标检测框架
- 使用Python 3.9开发
- 集成OpenCV、PyQt5、Ultralytics等库
- 提供图形用户界面
- 支持多平台环境（PyCharm+Anaconda或VSCode+Anaconda）
